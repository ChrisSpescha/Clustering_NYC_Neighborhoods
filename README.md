## Hello and Welcome!

The idea of this project is to segment the neighborhoods of New York City into separate clusters and examine the information about them.

### Files

- **data_preparation.ipynb** - contains information about data preparation/cleaning/EDA
- **modeling.ipynb** - contains information about the modeling part of the project

### Data

All info can be found on NYC Open Data.

I utilized info on Air Quality, Eviction Rate, and Crimes from 2022 in NYC neighborhoods

Hypothesis:
Mainly looking for relation between Air Quality and Crimes.
Find pattern within type of crimes and area.
Will look into Eviction Rate as extra Feature
